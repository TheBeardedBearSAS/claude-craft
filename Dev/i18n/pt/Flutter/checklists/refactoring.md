# Checklist de Refatoração Segura

## Preparação

- [ ] Todos os testes existentes passam
- [ ] Cobertura baseline registrada
- [ ] Branch de feature criada
- [ ] Backup/commit inicial feito

## Durante a Refatoração

- [ ] Uma modificação de cada vez
- [ ] Testes passam após cada etapa
- [ ] Commits frequentes
- [ ] Sem mudanças funcionais

## Verificações

- [ ] Todos os testes passam
- [ ] Cobertura mantida ou melhorada
- [ ] Sem regressão de performance
- [ ] flutter analyze limpo
- [ ] Compilação sem erros

## Antes do Merge

- [ ] Code review aprovado
- [ ] Documentação atualizada
- [ ] CHANGELOG atualizado
- [ ] Testes de integração passam
