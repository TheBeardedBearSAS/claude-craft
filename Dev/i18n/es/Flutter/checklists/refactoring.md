# Checklist Refactorización Segura

## Preparación

- [ ] Todas las pruebas existentes pasan
- [ ] Cobertura de referencia registrada
- [ ] Branch de funcionalidad creado
- [ ] Backup/commit inicial hecho

## Durante la Refactorización

- [ ] Una modificación a la vez
- [ ] Pruebas pasan después de cada paso
- [ ] Commits frecuentes
- [ ] Sin cambios funcionales

## Verificaciones

- [ ] Todas las pruebas pasan
- [ ] Cobertura mantenida o mejorada
- [ ] Sin regresión de rendimiento
- [ ] flutter analyze limpio
- [ ] Compilación sin errores

## Antes de Merge

- [ ] Code review aprobado
- [ ] Documentación actualizada
- [ ] CHANGELOG actualizado
- [ ] Pruebas de integración pasan
